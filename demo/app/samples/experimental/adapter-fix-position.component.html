Available since v1.2.4, undocumented

The first option of Adapter.fix method is scrollPosition. It allows to change the position of the viewport's scrollbar programmatically. It is used in the lib tests.

The value of scrollPosition should be an integer or Infinity. Negative value would be treated as zero. Despite the browsers have no agreement on how to deal with setting scrollTop/scrollLeft properties into +/-Infinity, the uiScroll treats them as follows:

  • scroll to the beginning of the viewport if -Infinity, (the value will be set to 0)
  • scroll to the end of the viewport if Infinity (scroll position will be set into the viewport's scrollHeight or scrollWidth value)
