{{item.text}} [remove]

Adapter.remove method allows to remove items from current uiScroll buffer. This method accepts 1 argument which is a predicate function applying to each item in the current buffer. If the return value is true, the item will be removed. In this demo we have pretty simple predicate: 

{{predicateDescription}}

Since the argument of the predicate has ItemAdapter type, the data contains id-text objects defined at the component level. The argument of doRemove method comes from the template as the id of the item the user clicked on.

The very important thing is to synchronize our datasource with the changes we are making over the uiScroll buffer via Adapter.remove. Generally this is the App component responsibility, and in this demo it happens via doRemoveDatasource(id) method. It removes item with given id from the initial dataset, and decrements id of all items whose ids are greater than the given one.
